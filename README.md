# To Do List

## Funcionalidades
- Criar tarefa
- Listar todas as tarefas
- Pegar tarefa em específico
- Marcar tarefa
- Excluir tarefa

## Tecnologias
### Backend
- Air
- Gin
- Go
- Gorm
- Sqlite3

### Frontend
- Axios
- Express
- Handlebars
- Javascript

## Como rodar
```bash
## Entre no projeto
cd todolist

## Entre em backend
cd backend

## Instale as dependências
go mod tidy

## Rode com air
air

## Saia da pasta backend
cd ..

## Entre na pasta frontend
cd frontend

## Instale as dependências
npm install

## Rode com npm
npm start
```

## Requisitos
- Air
- Go
- Npm
